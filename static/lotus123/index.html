Lotus 1-2-3
A1

Controls

  • Arrow Keys - Navigate cells
  • Enter/Return - Edit cell / Confirm entry
  • Escape - Cancel edit
  • F2 - Edit mode
  • Delete - Clear cell
  • Ctrl+C/V - Copy/Paste
  • Tab - Move to next cell

This is a Lotus 1-2-3 emulator running in your browser via DOSBox WebAssembly.
